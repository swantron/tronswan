# tronswan

.. a react app 
... with a cicd pipeline

## Scripts

run the following

### `npm start`

app runs in development mode at [http://localhost:3000](http://localhost:3000)

hot reloads and linting errors in the console

### `npm test`

interactive test runner

### `npm run build`

builds and bundles for DO deployment
